# Консольное приложение, эмулирующее вендинговый аппарат

Консольное приложение для симуляции работы вендингового аппарата.

## Что в себя включает

- Режим администратора и покупателя
- Управление продуктами
- Система оплаты монетами
- Отчет о прибыли

## Как запускать

```bash
git clone https://github.com/aiaanad/oop_sem1.git
cd oop_sem1/lab_1/src
dotnet build 
dotnet run ./Program.cs
